Title: Writing test failure (ERR_205)

Content:

If you get this alert when trying to install Bludit, you need to set the correct permissions to the `content` directory.

#### Content
1. [Hosting server change permissions via FTP](#ftp)
1. [Permissions on GNU/Linux](#linux)

---

## <a id="ftp"></a> Hosting server change permissions via FTP

If you have access to your Webserver via FTP, you can set the permission right-clicking on the directory/folder `content`, set permissions. First try to set the permission to 755, if the error continues then try with 777 permission.
