Title: My first theme
Content:

First, create the folder structure, in this example I will call my new theme `Coffee`.

1. Create the folder theme inside `/themes/` folder, you will get `/themes/coffee/`.
2. Create the folder `languages` inside `/themes/coffee/languages/`.
3. Create the file `index.php` inside `/themes/coffee/`, with the next HTML and PHP code:
<pre><code data-language="html"><!DOCTYPE html>
<html>
<head>
	<meta charset="UTF-8">
	<title>Bludit</title>
</head>
<body>
	<?php foreach ($posts as $Post): ?>

	<div><?php echo $Post->title() ?></div>
	<div><?php echo $Post->content() ?></div>

	<hr>

	<?php endforeach; ?>
</body>
</html>
</code></pre>

4. Create the file called `en_US.json` with the information about the theme, inside `/themes/coffee/languages/` folder.
<pre><code data-language="php">{
	"theme-data":
	{
		"name": "Coffee",
		"description": "This is my first theme for Bludit.",
		"author": "Diego",
		"email": "",
		"website": "http://www.bludit.com",
		"version": "1.0",
		"releaseDate": "2015-10-23"
	}
}
</code></pre>

5. Now you will see the theme on the **Dashboard->Themes**. This theme shows only the posts of the first page.
