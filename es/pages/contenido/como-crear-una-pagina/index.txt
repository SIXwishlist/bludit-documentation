Title: Como crear una pagina
Position: 2

Hay dos formas para crear una pagina. La primera que es la mas fácil, es utilizando la interfaz web; La segunda forma es editando el sistema de archivos.

### Indice
1. [Modo fácil](#facil)
1. [Modo avanzado o Cli mode](#avanzado-climode)

---

## <a id="facil"></a> Modo fácil
Ingrese al panel de administración, sobre el menú superior, haga clic en **Nueva pagina**.

Complete el formulario, ingrese un titulo para su nueva pagina y un contenido, luego haga clic en el botón **Guardar**. Eso es todo, ahora puede ingresar en su sitio y ver su nueva pagina.

![Bludit Como crear una nueva pagina](bludit-new-page.png)

<div markdown="1" class="note">
<div class="note-title">NOTA</div>
Bludit soporta contenido Markdown y HTML, aca puede encontrar una guía de código [Markdown](https://github.com/adam-p/markdown-here/wiki/Markdown-Cheatsheet).
</div>

## <a id="avanzado-climode"></a> Modo avanzado o Cli mode

Para utilizar este método es necesario activar el modo `Cli mode` y acceso a los archivos de Bludit vía SSH or FTP. Para activar el modo `Cli mode` vaya al panel de administración, sobre el menú superior **Configuración->Avanzado->Cli mode->Habilitar**.

- Crear un directorio dentro de `/bl-content/pages/`
- El nombre del directorio será el nombre de la URI que identificara a la pagina.
- Crear un nuevo archivo llamado `index.txt` dentro del nuevo directorio.
- Edite el contenido del archivo `index.txt`
- Agregue las dos variables necesarias para una pagina, `Title:` y `Content:`

<div markdown="1" class="note">
<div class="note-title">NOTA</div>
El nombre de las variables `Title:` y `Content:` tienen que estar en ingles, seguida de dos puntos.
</div>

### Ejemplo
- Creamos el directorio `animales` dentro de `/bl-content/pages/`
- Creamos el archivo `index.txt` dentro de `/bl-content/pages/animales/`
- Editamos el archivo `/bl-content/pages/animales/index.txt`
- Agregamos el siguiente texto

```
Title: Mi nueva pagina de animales

Content:
<h1>Esto es h1 en HTML</h1>
```
