Title: Installationsanleitung
Position: 2
Content:

1. Lade die neueste Version von Bludit von http://www.bludit.com/bludit_latest.zip herunter.
2. Entpacke die ZIP-Datei in ein Verzeichnis, dass du beispielsweise `bludit` benennst.
3. Lade das Verzeichnis `bludit` mit FTP auf deinen Server.
4. Rufe im Browser die Domain mit der Installation auf, beispielsweise `http://domain.com/bludit`.

_Hinweis: _Note: Um die Dateien auf den Server zu laden, kannst Du einen FTP-Client verwenden. Vgl. dazu [FTP](../verschiedenes/ftp)._
