Es gibt zwei Möglichkeiten, einen Beitrag zu schreiben. Einen "Easy Mode" mit einer grafischen Oberfläche und einen "Advanced Mode" oder CLI-Mode (Command-line Interface), also Kommandozeilen-Modus. Wenn du den "Advanced Mode" verwenden möchtest, schau dir bitte die Erklärungen [Variablen für Beiträge und Seiten](../content/variables-for-posts-and-pages) an.

## Easy Mode <a id="easy-mode"></a>
Melde dich im Admin-Bereich an und wähle `Neuer Beitrag` in der horizontalen Menüleiste oder auf der Dashboard-Seite.

Fülle die Felder aus und klicke den Button  **Speichern**.

Zur Formatierung des Inhalts kannst du HTML- und Markdown-Code verwenden. Hier findest du ein [Markdown-Cheatsheet](https://github.com/adam-p/markdown-here/wiki/Markdown-Cheatsheet).

## <a id="advanced-mode"></a> Advanced Mode oder CLI Mode (Kommandozeilen-Modus)

Aktiviere zuerst den CLI Mode im Admin-Bludit von Bludit unter Einstellungen -> Erweitert -> Kommandozeilen-Modus -> CLI Mode: Aktiviert. Jetzt kannst Du einen Beitrag erstellen, ändern oder löschen, indem du direkt das Dateisystem bearbeitest, also ohne den Editor des Dashboards zu verwenden.

Beispielsweise kannst du einen neuen Beitrag so erstellen:

- Erstelle ein Unterzeichnis im Verzeichnis `/content/posts/`.
- Der Ordnername wird die URI des neuen Beitrags sein.
- Erstelle eine neue Datei mit dem Namen `index.txt` im neuen Verzeichnis.
- Bearbeite die Datei `index.txt`.
- Füge zwei Variablen hinzu: `Title:` und `Content:`.

### Beispiel
- Erstelle das Verzeichnis `/content/posts/test`.
- Erstelle die Datei `index.txt` in `/content/posts/test/`.
- Bearbeite die Datei `index.txt`.
- Gib folgenden Text ein:

```
Title: Neuer Beitrag-Test

Content:
```
